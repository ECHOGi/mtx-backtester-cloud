# V083_CHANGELOG

## 版本
- 平台：v0.8.3
- 名稱：未來情境＋正二基準＋說明版

## 本次新增

### 1. 00631L正二實際基準
- 新增 `benchmark_00631l.py`。
- 可從TWSE官方月資料端點逐月下載00631L日行情並建立本機快取。
- 可改用使用者上傳CSV；至少需要日期與收盤價。
- 使用相同初始資金，以整數股數買進持有，剩餘資金保留現金。
- 匯出正二歷史資料、期末資產、年化報酬與最大回撤率。

### 2. 2026年1拆22分割處理
- 依本案已確認事件表：2026-03-31恢復交易，1單位分割為22單位。
- 原始價格資料：分割日自動將持有股數乘22，總資產保持連續。
- 報酬資料：分割日前價格除以累積分割比例，避免產生約-95%的假暴跌。
- 若來源已是調整後價格，會檢查分割日前後價格比例，避免重複調整。
- 暫停交易期間沒有價格時，不會補成0；持有部位延用最後可用估值。

### 3. 多截止日＋未來日K情境
- 新增 `future_scenarios.py`。
- 可選多個共同歷史截止日。
- 每個截止日後接上六種共同未來狀態：
  1. 低波動上漲
  2. 高波動上漲
  3. 低波動盤整
  4. 高波動震盪
  5. 緩慢下跌
  6. 崩跌後反彈
- 情境由歷史日K的報酬、波動、跳空、上下影線與成交量區段抽樣，不依策略績效挑選參數。
- 三個策略與00631L共用相同來源日期與seed。
- 延伸長度採歷史持倉期第99百分位×2，自動限制在126～504個交易日。

### 4. 正式情境排名
- 新增：
  - 已實現損益中位數、P25、P10
  - 已實現年化報酬中位數
  - 最大回撤率中位數及最差值
  - 斷頭路徑比例
  - 尚未自然出場比例
  - 超越00631L正二路徑比例
  - 相對正二年化差
- 排名順序：超越正二路徑比例 → 已實現年化中位數 → 已實現損益P10。

### 5. 主畫面與「？」說明
- 左側欄位補回help說明：研究模式、資料週期、盤中模擬情境數、部位模式、初始資金、安全資金、正二來源、費率、滑價、斷頭檢查、seed、共同截止日與未來路徑數。
- 主畫面加入「如何閱讀本頁結果」。
- 純日K單次結果自動隱藏沒有意義的P25、P10、獲利路徑比例、代表seed與箱型圖。
- 新增顯示：扣除期末強平後損益、最大回撤率、最大有效槓桿、00631L績效與情境比較。

### 6. 大型複利部位效能修正
- 安全資金部位由逐單位遞減改為二分搜尋。
- 帳戶複利成長後即使理論曝險達數百或數千微台等值，也不會因逐口測試而卡住。
- 仍維持大台→小台→微台的最少契約口數換算。
