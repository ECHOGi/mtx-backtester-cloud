# v0.8.8.0 更新紀錄

## 新增：50萬元滾動起點敏感度

- 每月第一個交易日建立全新50萬元空手帳戶。
- 不繼承持倉、待執行訊號、冷卻、布林鎖、回撤煞車或交易狀態。
- 支援啟動後1年、2年、3年多期限觀察。
- 依起點前一根已完成日K分類穩定牛市、高波動牛市、穩定熊市、高波動熊市／恐慌、低波動盤整及高波動盤整。
- 分開統計維持保證金強制平倉與帳戶權益小於等於0。
- 輸出期末資產分位數、資產門檻比例、最低權益、最大回撤、回本時間及第一筆交易等待時間。

## 新增：同日起點50萬元ALL-IN 00631L

- 每個起點及每個期限，以50萬元按00631L起點當日或其後第一個可交易日收盤價買進整數股。
- 買不到整數股的剩餘現金留在帳戶。
- 持有至與策略相同結算日，分割事件自動調整股數。
- 輸出買進日、價格、股數、手續費、剩餘現金、期末價格、期末資產、最低資產與最大回撤。
- 新增正二逐起點、正二市場狀態彙總及策略對正二金額比較三份報表。

## 正確性修正

- 起點前高優先空單信號不再污染起點後的10日封鎖狀態。
- L16與L14在同一起點保持完全相同的訊號與進出場骨架，只允許口數與資金結果不同。
- 資料不足時只略過不足的較長期限，不刪除可用的短期限結果。
