# v0.8.8.2 更新紀錄

## 新增：L14→L16已實現權益門檻切換

- 新增 `use_equity_risk_switch` 與低／高風險率、升級／降級門檻設定。
- 門檻只在帳戶空手、下一筆新進場執行前判斷。
- 判斷基礎固定為已實現帳戶權益，不使用盤中未實現損益。
- 持倉期間不變更口數，也不追溯修改既有交易。
- 升級後可維持L16，或設定低於降級門檻時回到L14，形成遲滯區間。
- 空單仍沿用原S10安全資金部位規則，不受L14／L16切換影響。

## 新增報表

- `19_權益門檻切換_逐起點明細.csv`
- `20_權益門檻切換_策略彙總.csv`

報表包含首次升級日期、升級等待交易日、升降級次數、L16運作占比、升級後最低權益，以及升級後是否跌破原始50萬元。
