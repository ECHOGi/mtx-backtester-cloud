# v0.4.8 更新說明｜樣本內／樣本外自動驗證

## 本版目的

避免每次人工分別跑樣本內、樣本外後，再用肉眼比對兩張批次比較表。

## 新增功能

### 1. 樣本內／樣本外自動驗證按鈕

在批次策略回測區新增：

- `🧪 樣本內／樣本外自動驗證`
- `🧪 手動 JSON 樣本內／外驗證`

平台會用同一份批次 JSON 自動跑兩段期間：

- 樣本內：2015-01-01 ～ 2023-12-31
- 樣本外：2024-01-01 ～ 資料最後一日

### 2. 自動產出樣本內外對照表

新增輸出檔：

- `sample_in_vs_out_comparison.csv`
- `00_樣本內外驗證總覽.md`

主要欄位：

- 樣本內報酬回撤比
- 樣本外報酬回撤比
- 樣本外／樣本內比
- 樣本內總損益
- 樣本外總損益
- 樣本內最大回撤
- 樣本外最大回撤
- 樣本外獲利因子
- 樣本外期望值
- 樣本外交易次數
- 判讀

### 3. 判讀標籤

平台會自動給出初步判讀：

- 穩定保留
- 樣本外爆發，需複查
- 可觀察
- 樣本外退化
- 偏弱
- 樣本外失敗

## 輸出結構

樣本內／外驗證 ZIP 與自動保存資料夾會包含：

```text
00_樣本內外驗證總覽.md
sample_in_vs_out_comparison.csv
sample_in/
  00_批次回測總覽.md
  batch_comparison.csv
  各策略資料夾...
sample_out/
  00_批次回測總覽.md
  batch_comparison.csv
  各策略資料夾...
```

## 使用方式

1. 將批次 JSON 放入 `_策略投放箱`。
2. 在網站選擇該 JSON。
3. 直接按 `🧪 樣本內／樣本外自動驗證`。
4. 看畫面上的「樣本內／樣本外驗證結果」。
5. 平台會自動保存到 `MTX Test Record`。

## 注意

本功能是診斷工具，不會自動決定下一組策略。若出現「樣本外爆發，需複查」，代表可能吃到特定年度行情，需要再做第二階段細部測試。
