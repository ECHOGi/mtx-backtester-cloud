# v0.5.3 改版說明

## 改版日期
2026-07-10

## 改版重點

### 1. 雲端策略連結加入內建備援

v0.5.2 在新電腦上可使用雲端策略連結，但 Google Drive 分享頁有時會回傳 HTML 網頁，而不是 JSON 原始檔，導致平台顯示：

> 下載到的是網頁，不是 JSON。

v0.5.3 修正為：

- 優先讀取雲端策略 JSON 連結。
- 如果雲端下載失敗、下載到 HTML、或 JSON 驗證失敗，平台自動改用內建 `batch_009_獲利放大同步放大出場條件診斷.json`。
- 使用者不需要安裝 Google Drive 桌面版，也不需要手動下載策略 JSON。

### 2. 新增內建策略資料夾

新增：

```text
bundled_strategies/
└─ batch_009_獲利放大同步放大出場條件診斷.json
```

這批策略專門測試：

- 03 系列基準策略
- 最高浮盈分段吊燈
- 獲利達門檻後排除 MACD 反向出場
- 獲利放大同步放大出場條件

### 3. 操作流程

使用者在新電腦上只需要：

1. 開啟 v0.5.3。
2. 批次策略回測區使用預設「雲端策略連結」。
3. 選「前後期行情對照」。
4. 按「開始雲端批次回測」。

若雲端讀取失敗，平台會自動顯示警告並使用內建 batch_009 繼續執行。

## 技術調整

- 新增 `APP_DIR`、`BUNDLED_BATCH_009_FILENAME`、`BUNDLED_BATCH_009_PATH`。
- 新增 `load_bundled_batch_009_json()`。
- 修改雲端策略按鈕流程：雲端讀取失敗時不再中止，改用內建策略。
- 保留本機策略投放箱作為備用。

## 驗證

- `app.py` 已通過 Python 語法編譯檢查。
