# v0.4.9 變更紀錄｜前後期行情對照 UI 修正

建立日期：2026-07-09

## 本版目的

修正 v0.4.8 批次回測區的操作混淆問題。

原本同時存在：

- 批次回測
- 樣本內／樣本外自動驗證

對非工程使用者來說像是「沒有選擇的選擇」，而且「樣本內／樣本外」名稱容易誤會。

本版改成以行情階段為中心：

- 前期行情：2015～2023，一般行情／盤整震盪期／非大牛市期
- 後期牛市：2024～資料末日，強趨勢行情／牛市期

## 修改內容

### 1. 批次回測區改為單一模式選擇

策略投放箱與手動 JSON 都改為先選「批次回測模式」，再按一顆主按鈕。

模式包含：

1. 前後期行情對照：2015～2023 一般行情 vs 2024～資料末日牛市行情
2. 前期行情回測：2015～2023
3. 後期牛市回測：2024～資料末日
4. 目前畫面期間回測：使用上方起迄日

### 2. 移除容易混淆的雙按鈕流程

不再同時顯示：

- 讀取投放箱並批次回測
- 樣本內／樣本外自動驗證

改為：

- 選擇模式
- 按「開始批次回測」

### 3. 名稱改為「前後期行情對照」

原本的：

- 樣本內
- 樣本外
- 樣本內外驗證

改為：

- 前期行情
- 後期牛市
- 前後期行情對照

### 4. 輸出檔名同步調整

新增／調整輸出：

- `market_phase_comparison.csv`
- `00_前後期行情對照總覽.md`
- `phase_2015_2023/`
- `bull_2024_plus/`

### 5. 對照表欄位改名

主要欄位改為：

- 前期行情報酬回撤比
- 後期牛市報酬回撤比
- 後期/前期比
- 前期行情總損益(元)
- 後期牛市總損益(元)
- 前期行情最大回撤(元)
- 後期牛市最大回撤(元)
- 行情適應判讀

### 6. 判讀文字改為行情語言

原本：

- 樣本外爆發，需複查
- 樣本外退化
- 樣本外失敗

改為：

- 前後期皆穩定
- 前期普通，牛市強勢
- 前期穩定，後期失效
- 後期牛市失效
- 可觀察
- 偏弱
