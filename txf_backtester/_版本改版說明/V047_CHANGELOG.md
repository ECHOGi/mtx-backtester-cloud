# v0.4.7 改版重點

1. 比較表欄位中文化：`Profit Factor` 改為 `獲利因子`。
2. `最長回撤持續(K棒)` 改名為 `資金持續未創新高交易天數`，避免誤解為單筆持倉時間。
3. 批次比較表欄位順序改為判讀優先序：報酬回撤比、總損益、最大回撤、獲利因子、期望值、交易次數、勝率、最大連續虧損、平均持倉、資金持續未創新高交易天數。
4. 批次比較表預設依判讀優先序排序：報酬回撤比由高到低、總損益由高到低、最大回撤絕對值由低到高、獲利因子由高到低。
5. 將根目錄歷次改版說明與測試報告移至 `_版本改版說明/`，降低主程式資料夾雜亂。
