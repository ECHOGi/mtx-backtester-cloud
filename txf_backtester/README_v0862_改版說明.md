# v0.8.6.2｜200筆檢查點＋回撤煞車觀察修正版

## 本次小幅改版

### 1. 長回測檢查點由500筆改為200筆

- 多截止日未來情境回測每完成200筆即寫入檢查點。
- 正常完成或程式可攔截的中斷仍會補寫最後不足200筆的尾批。
- 若雲端主機被直接強制終止，最多重算最近199筆；已落盤結果仍可接續。
- 檢查點識別包含平台版本，因此不會把v0.8.6.1舊檢查點誤接到新版結果。

### 2. 新增回撤煞車逐日觀察欄位

代表路徑資金曲線新增：

- `daily_drawdown_brake_multiplier`
- `daily_drawdown_brake_active`
- `daily_realized_equity_drawdown_pct`
- `daily_realized_equity_peak`

這些欄位表示當日結束後，下一筆新進場會採用的煞車狀態。計算只使用已實現權益，與實際部位計算規則一致，不會以未實現浮盈虧損誤觸發，也不會在持倉途中變更既有口數。

### 3. 批次結果新增煞車統計

歷史比較、未來情境完整分布、策略比較與市場狀態比較新增：

- 回撤煞車觸發次數
- 煞車狀態交易日數
- 煞車狀態交易日占比
- 平均每日回撤煞車倍率
- 最低每日回撤煞車倍率

用途是區分：

1. 煞車偶爾啟動並精準避開尾端風險。
2. 煞車長期常駐，實際上只是變相降低風險預算。

## 相容性

- 策略JSON格式不變。
- v0.8.6.1策略可直接使用。
- 回測、正二比較、續跑與雲端上傳流程不變。
