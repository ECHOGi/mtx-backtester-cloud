# V084_CHANGELOG

## 版本
- 平台：v0.8.4
- 名稱：期末總權益正二對照＋NX回撤煞車版

## 本次改版原則
- 未來永遠持續，因此「模擬終點仍有未平倉部位」不是錯誤，也不再作為排名扣分項目。
- 正式主排名改用共同截止日、共同未來狀態與共同seed下的「策略期末總權益」直接比較「00631L期末市值」。
- 已實現損益與尚未自然出場比例仍保留，僅供判讀交易完成度與策略持倉特性。

## 本次新增

### 1. 主排名改為期末總權益對正二期末市值
多截止日未來情境結果新增並列顯示：
- 策略期末總權益中位數、P25、P10
- 00631L期末資產中位數、P25、P10
- 期末資產差中位數、P10
- 期末總權益超越正二比例
- 策略總權益年化報酬率與正二年化報酬率
- 相對正二年化差
- 策略最大回撤率與正二最大回撤率
- 相對正二回撤改善
- 斷頭比例、尚未自然出場比例及已實現損益

正式排序順序：
1. 期末總權益超越正二比例
2. 相對正二年化差中位數
3. 期末資產差P10

### 2. 正二相對數值直接放入主表
- 歷史回測主表直接並列策略與00631L的期末資產、年化報酬及最大回撤率。
- 未來情境主表直接並列策略與00631L的P50、P25、P10及相對差值。
- 使用者不需再跨表換算即可判斷「多賺多少／少賺多少」及「少承擔多少回撤」。

### 3. 未來情境每種最多20條路徑
- 控制項選項擴充為1、2、3、5、10、15、20。
- 每個策略與00631L使用完全相同的截止日、未來狀態與seed。
- 尚未自然出場比例僅作資訊，不要求模擬一定要強制等待自然出場。

### 4. 新增已實現權益回撤煞車
為支援NX01、NX02，動態風險複利新增：
- `use_drawdown_risk_brake`
- `position_drawdown_brake_start_pct`
- `position_drawdown_brake_full_pct`
- `position_drawdown_brake_floor`

運作方式：
- 依「已實現權益高點」計算已實現回撤。
- 回撤達啟動門檻後，風險預算線性降低。
- 達完整煞車門檻後，降至指定最低倍率。
- 回撤恢復後可隨已實現權益重新增加風險。
- 仍受保證金、壓力資金、停損風險及斷頭檢查限制。

交易明細與績效新增：
- 基礎／實際風險比例
- 回撤煞車倍率
- 進場時已實現權益回撤
- 已實現權益高點
- 最低與平均煞車倍率
- 煞車啟動交易數

### 5. 動態風險模式支援「0＝無人為口數上限」
- `position_max_micro_units = 0` 不再被視為零口。
- 實際口數由風險預算、停損距離、帳戶權益、保證金與壓力檢查共同決定。
- 契約仍自動換算為大台→小台→微台的最少契約口數組合。

## batch_026策略組成
正式批次：
`batch_026_CC03_GE01_CC04_NX01_NX02_期末總權益正二主排名.json`

包含：

2. GE01_04：安全約束動態複利
3. CC04波動壓縮突破：保留原進出場邏輯，部位統一改為安全約束動態複利
4. NX01低波突破回撤煞車：保留動態風險複利與攻擊型回撤煞車
5. NX02雙向趨勢獲利鎖：保留多空邏輯、動態風險複利與均衡型回撤煞車

### 公平性處理
- 不納入任何固定1口、固定2口或人為最佳口數版本。
- CC04原檔的主力／穩健差異主要是固定口數，因此只保留一組進出場邏輯，改用與CC03、GE01相同的安全約束動態複利。
- NX01、NX02的回撤煞車屬於策略核心，因此保留其動態風險部位模型；本批比較的是完整交易系統，結果表會同步揭露回撤、斷頭與正二相對績效。

## 已知限制
- 未來日K情境是歷史條件式抽樣，不是市場預測。
- 20條路徑是研究取樣上限，不代表涵蓋所有可能未來。
- 未平倉部位會以模擬終點市值計入期末總權益。
- 換月價差與換月滑價仍未獨立模擬。
- 未來情境模式目前只支援完整日K策略。
